## Overview

`/scripts` contains utility scripts for development and project management.

There are these scripts:

- [check.sh](../scripts/check_README.md): The main quality control script. Runs all tests, linters, and formatters.
- [loc-counter](../scripts/loc-counter/README.md): Generates a CSV showing lines of code evolution over time.
- [roadmap-burndown](../scripts/roadmap-burndown/README.md): Generates a CSV burndown chart from `ROADMAP.md` history.
